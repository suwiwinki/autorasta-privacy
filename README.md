    <h2>Recopilación de Datos</h2>
    <p>La extensión autorasta NO recopila, almacena ni transmite ningún dato personal del usuario.</p>
    
    <h2>Almacenamiento Local</h2>
    <p>La extensión utiliza el almacenamiento local del navegador (chrome.storage) únicamente para guardar las preferencias de configuración del usuario (accesos rápidos, atajos de teclado, etc.). Esta información permanece exclusivamente en el dispositivo del usuario.</p>
    
    <h2>Permisos</h2>
    <p>La extensión requiere los siguientes permisos:</p>
    <ul>
        <li><strong>activeTab:</strong> Para interactuar con las páginas de workcrm.com</li>
        <li><strong>storage:</strong> Para guardar la configuración del usuario localmente</li>
        <li><strong>scripting:</strong> Para inyectar funcionalidades en workcrm.com</li>
        <li><strong>tabs:</strong> Para navegación y redirecciones</li>
    </ul>
    
    <h2>Dominio</h2>
    <p>La extensión solo funciona en el dominio workcrm.com y no accede a otros sitios web.</p>
