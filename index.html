Política de Privacidad - autorasta

Última actualización: 18 de noviembre de 2024

1. Recopilación de Datos

La extensión autorasta NO recopila, almacena ni transmite ningún dato personal del usuario a servidores externos.

No recopilamos información personal identificable, datos de navegación, ni ningún otro tipo de información sensible.

2. Almacenamiento Local

La extensión utiliza el almacenamiento local del navegador (chrome.storage) únicamente para guardar las preferencias de configuración del usuario, incluyendo:

  • Accesos rápidos personalizados
  • Atajos de teclado configurados
  • Preferencias de automatizaciones
  • Configuraciones de popups y alertas

Esta información permanece exclusivamente en el dispositivo del usuario y nunca se transmite a servidores externos.

3. Permisos de la Extensión

La extensión requiere los siguientes permisos para funcionar correctamente:

  • activeTab: Para interactuar con las páginas de workcrm.com y proporcionar las funcionalidades mejoradas.
  • storage: Para guardar la configuración del usuario localmente en su navegador.
  • scripting: Para inyectar funcionalidades y mejoras en las páginas de workcrm.com.
  • tabs: Para navegación y redirecciones automáticas según las preferencias del usuario.

Todos estos permisos se utilizan únicamente para mejorar la experiencia del usuario dentro de workcrm.com.

4. Dominio de Operación

La extensión solo funciona en el dominio workcrm.com y no accede, modifica ni interactúa con ningún otro sitio web.

No se ejecuta en otras páginas web fuera de workcrm.com.

5. Cookies y Tecnologías de Seguimiento

La extensión no utiliza cookies ni tecnologías de seguimiento. No rastreamos la actividad del usuario ni recopilamos datos analíticos.

6. Compartir Información

No compartimos, vendemos ni transferimos información del usuario a terceros, ya que no recopilamos ningún tipo de información.

7. Seguridad

Toda la configuración se almacena localmente en el navegador del usuario utilizando las APIs seguras proporcionadas por Chrome. No hay transmisión de datos a servidores externos.

8. Cambios en esta Política

Nos reservamos el derecho de actualizar esta política de privacidad. Cualquier cambio será reflejado en la fecha de "Última actualización" al inicio de este documento.
